HTML Encoder/Decoder

Encode atau decode HTML code Anda dengan berbagai metode

Informasi Metode Encoding:
  • Base64: Encoding standar yang aman, cocok untuk menyimpan data dalam format teks. Paling umum digunakan.
  • Hexadecimal: Mengkonversi setiap karakter ke format hex (0-9, A-F). Bagus untuk debugging.
  • URI Component: Encoding untuk URL, mengubah karakter khusus menjadi format %XX.
  • HTML Entities: Mengubah karakter HTML khusus menjadi entities (<, >, &, dll).
